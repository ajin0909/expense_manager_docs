expense_manager_library documentation
=====================================
expense_manager_library is a Python library that analyzes an individual's consumption data and generates reports. You can use this library to generate PDF reports including consumption amounts by category, monthly consumption history, and graphs.

**KEY FEATURES**

- **Consumption Data Analysis**: Import consumption data from a CSV file and automatically categorize it.
- **Graph Creation**: Provides a variety of visualizations, including pie charts, line charts, bar charts, and more.
- **Calculate sum and percentage of consumption by category**: Extract key consumption patterns from your data.
- **Monthly Consumption Analysis**: Analyze the monthly consumption amount and category percentage.
- **Generate PDF Report**: Save your expense report, including graphs and data, as a PDF.
